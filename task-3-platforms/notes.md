## Task 3 Write-up: Coding & Collaboration Platforms

For Task 3, I explored both coding practice and cloud collaboration platforms to enhance my technical and organizational skills. 

For the coding segment, I chose HackerRank and completed the beginner-level Python introduction challenges. I selected HackerRank because of its structured 
progression, which perfectly complements my current focus on mastering Python syntax and logic. Practicing on this platform helps solidify the foundational 
programming skills I need as I progress deeper into my Artificial Intelligence and Machine Learning coursework. 

For the collaboration segment, I utilized Google Forms to build a 5-question "Digital Literacy Awareness Quiz," incorporating both multiple-choice 
and short-answer formats to test my peers' knowledge. Creating this form gave me practical experience in designing digital surveys and managing response 
data seamlessly via Google Sheets.

Academically, these tools are invaluable. HackerRank provides the continuous algorithmic practice essential for technical interviews and complex problem-solving. 
Meanwhile, mastering Google Workspace tools like Forms and Sheets prepares me for efficient data collection, user surveys, and seamless team collaboration in future 
engineering projects.
